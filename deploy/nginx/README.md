# Nginx hardening (template)

Этот конфиг — шаблон. Он **не применяется автоматически**.

## Что делает
- Прячет `Server`/`X-Powered-By`
- Добавляет базовые security‑headers
- Проксирует на локальный gunicorn
- Настраивает кеширование статических ассетов

## Как использовать (вручную)
1) Скопировать `dance_studio_hardening.conf` в nginx sites‑available.
2) Подставить свои домены/пути/сертификаты.
3) Проверить `nginx -t`.
4) Перезапустить nginx.

## Важно
Не применять без теста на staging.
